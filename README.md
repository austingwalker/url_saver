# URL_Saver

## Deployed app at: 

https://peaceful-tundra-98649.herokuapp.com/

**Description:**
    URL_Saver is a full-stack application created to keep track of URL's. A common occurence is copy and pasting a url into the notes section of one's iphone and then never seeing it again. It uses a MySQL database and the one-to-many table relationship to allow the user to create a file and save URL's to it for quick and easy access later. 
    
## Technologies used: 
    React, Javascript, JSX, Bootstrap, Axios, Node.js, Express, MySQL, Sequelize
